# 🪔 Diwali Sales Data Analysis

## 📌 Project Overview
This project analyzes Diwali sales data to understand customer purchasing behavior and identify key trends that can help businesses improve their marketing strategies during festive seasons.

The analysis focuses on data cleaning, exploration, and visualization to extract meaningful insights from raw sales data.

---

## 🛠️ Tools & Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## 📂 Dataset
- Dataset: Diwali Sales Data  
- Format: CSV file  
- Contains customer demographics, product categories, and purchase details  

---

## 🔍 Project Workflow

### 1. Data Collection
- Imported dataset using Pandas  
- Loaded CSV file with proper encoding  

### 2. Data Cleaning
- Checked dataset structure using `.info()`  
- Removed unnecessary columns (e.g., columns with all NULL values)  
- Handled missing values  
- Cleaned and prepared data for analysis  

### 3. Data Exploration
- Analyzed dataset using:
  - `.head()` for preview  
  - `.shape` for size  
  - Summary statistics  

### 4. Data Visualization
- Used Matplotlib and Seaborn to create visual insights  
- Plotted graphs for:
  - Gender distribution  
  - Age group analysis  
  - State-wise sales  
  - Occupation-based purchasing  
  - Product category performance  

---

## 📊 Key Insights

- Female customers contributed more to overall sales  
- Age group 26–35 showed the highest purchasing activity  
- Married individuals had higher purchasing power  
- Customers from specific states contributed significantly to revenue  
- Certain product categories performed better during Diwali season  

---

## 📈 Conclusion
This analysis helps businesses understand their target customers and optimize their marketing strategies during festive seasons like Diwali.  

The insights can be used for:
- Targeted advertising  
- Product recommendations  
- Inventory planning
